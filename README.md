# data-analytics-intern-assignment

## Overview
This project is a JavaScript-based data analysis solution built using the provided Excel dataset.

## Features Implemented
- Dynamic aggregation by time granularity (Daily / Weekly / Monthly)
- Correct entity-level deduplication (unique customers by phone)
- JavaScript-only data processing (no manual Excel changes)
- Dynamic bar chart visualization
- Cross-table matching between Orders and Calls
- Derived metric: Orders per Call

## How to Run
1. Download the repository
2. Open `index.html` in a browser
3. Select time granularity
4. Upload the provided Excel file
5. View chart and console output

## Notes
All data preparation, aggregation, and matching logic is implemented in JavaScript as per assignment requirements.
